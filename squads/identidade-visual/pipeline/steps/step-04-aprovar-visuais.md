# Checkpoint: Aprovar Visuais

Revisar os conceitos visuais produzidos por Vitor Visual.

**Leia:** `squads/identidade-visual/output/visuais-aprovados.md`

**Aprove** se:
- 3 conceitos de logo foram produzidos com abordagens genuinamente distintas (não variações do mesmo)
- Nenhum conceito usa símbolo proibido: árvore, coração, figura humana literal, cruz vermelha
- A paleta tem pelo menos 4 cores com HEX especificado e ratios de contraste WCAG documentados
- O par tipográfico está especificado com hierarquia de pelo menos 4 níveis
- Pelo menos 2 mockups foram renderizados mostrando logo + paleta + tipografia integrados
- Os mockups parecem pertencer à mesma marca (coerência visual)
- A identidade é claramente distinta dos concorrentes investigados: @instituto_singular (teal), centrobaoba (verde + árvore)

**Rejeite** se:
- Qualquer conceito de logo usa símbolo genérico proibido — indicar qual e solicitar substituto
- A paleta usa azul hospitalar genérico ou teal (#00B4C5) já ocupado por concorrente
- Os 3 conceitos de logo são variações do mesmo elemento (sem diversidade de abordagem)
- Não há ratios de contraste WCAG documentados

Digite **aprovar** para continuar com Carla Compilação no brand guide, ou indique o que deve ser revisado por Vitor Visual.
